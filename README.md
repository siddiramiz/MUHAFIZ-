MUHAFIZ is an emergency alert and disaster management system designed to assist users during fire and emergency incidents.

## Features (MVP)
- One-tap SOS
- Live location sharing
- Emergency alerts
- Mark Safe / Unsafe
- Emergency contacts
- Offline safety instructions
- Admin demo view

## Tech Stack
- Frontend: React / React Native
- Backend: Node.js
- Database: Firebase Realtime Database
- Maps: Google Maps API
- Notifications: Firebase Cloud Messaging
